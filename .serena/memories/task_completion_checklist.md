# タスク完了時のチェックリスト

## 必須項目
1. **型チェック**: `npx tsc --noEmit` でエラーがないこと
2. **動作確認**: `npm run dev` で実際に動作を確認
3. **コメント追加**: 新規コンポーネントや重要ロジックに JSDoc コメントを追加
4. **既存機能の退行確認**: 既存の機能が壊れていないか手動確認

## 推奨項目
1. **リントチェック**: 利用可能であれば `npm run lint` を実行
2. **コード整形**: 一貫したコードスタイルを維持
3. **ドキュメント更新**: 必要に応じて docs/ 以下のチケットやガイドを更新
4. **状態管理の確認**: AppStateContext への変更がある場合、依存コンポーネントへの影響を確認

## コミット前
- 不要なコンソールログやデバッグコードを削除
- 未使用のインポートを削除
- ファイルの末尾に改行があることを確認
